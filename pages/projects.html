Projects

I began making these landing pages while working on Codecademy's HTML/CSS course. I have since rewritten most of them to remove the reliance on frameworks and improve the responsiveness of each page. I have included a variety of techniques such as CSS Grid and Flexbox, and the CSS code uses SASS nesting and variables.

Mirae English

Our school's homepage has been created using CSS Grid and Flexbox and Responsive Design using media queries. The information section features an embedded Google Map showing the location.

Darkside

This page is an example of a fully responsive design incorporating a Javascript-enhanced submission form.

Twin Studio

This uses older techniques of inline-block and auto margin to style the photos responsively.

Sleep

While the outline design of this page was taken from a Codecademy project, the code itself has been completely rewritten to give a responsive website using Flexbox.

Halloween

This responsive page relies on Flexbox to easily align the items. Javascript was used to scroll through the photos and the background image was darkened by using an opacity value over a black background.

Guam

This was an early attempt to use CSS Grid and is not responsive.
